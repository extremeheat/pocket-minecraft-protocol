Login Sequence

The login process is as follows:

  • C‚ÜíS: Login
  • S‚ÜíC: Server To Client Handshake
  • C‚ÜíS: Client To Server Handshake
  • S‚ÜíC: Play Status (Login success)
  • To spawn, the following packets should be sent, in order, after the ones above:
  • S‚ÜíC: Resource Packs Info
  • C‚ÜíS: Resource Pack Client Response
  • S‚ÜíC: Resource Pack Stack
  • C‚ÜíS: Resource Pack Client Response
  • S‚ÜíC: Start Game
  • S‚ÜíC: Creative Content
  • S‚ÜíC: Biome Definition List
  • S‚ÜíC: Chunks
  • S‚ÜíC: Play Status (Player spawn) If there are no resource packs being sent, a Resource Pack Stack can be sent directly after Resource Packs Info to avoid the client responses.
